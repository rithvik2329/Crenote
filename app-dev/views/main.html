Oops !

Don't panic ;)

There has been a file reading or writing error.
It means that can't read the content of one or several of its files, edit them or create them.
Don't worry we will find a solution.

This error can appear at the application startup, when you try to modify a setting, or save a document with a theme.

In these cases, encountering this error means that is probably installed in a restricted location on your computer.
The easy and recommended way to solve this error is to uninstall the application and reinstall it into a more common location.

On windows, you can ALTERNATIVELY run as administrator if the user account you are using allows you to do so.

This error can also appear when you try to open, or save a document, with or without a theme.

In these cases, encountering this error means that the documents you're trying to save or open, are probably located in a restricted location on your computer.
The easy and recommended way to solve this error is to choose another saving location or move the documents you're trying to open and retry opening them.
